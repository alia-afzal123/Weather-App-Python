# 🌤️ Weather App

A simple and user-friendly desktop weather application built with Python. Get real-time weather information for any city worldwide.

## ✨ Features

- 🌍 **Real-time weather data** from WeatherAPI.com
- 🌡️ **Temperature in Celsius / Fahrenheit** (toggle between units)
- 💧 **Humidity percentage** display
- 💨 **Wind speed** in km/h
- ☀️ **Weather conditions** (Sunny, Rainy, Cloudy, etc.)
- 🎨 **Clean and intuitive GUI** using Tkinter
- ⚡ **Fast response** with error handling

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Python 3.x | Core programming language |
| Tkinter | GUI development |
| Requests | API calls |
| WeatherAPI.com | Weather data provider |
